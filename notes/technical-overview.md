This repo is all about building a well-organized Everyday Carry (EDC) pouch. It aims to create a compact kit to hold essential tools for daily use.

### Features
- Modular pouch design
- Curated tool list
- Justification for included items
- Visuals of the pouch and contents

### Tech Stack
- Markdown for documentation
- Images for visual context

### Getting Started
No installation needed. Just clone the repo and check it out locally:

```bash
git clone https://github.com/justin-napolitano/curate-an-edc-pouch.git
cd curate-an-edc-pouch
open index.md
```

### Project Structure
```
curate-an-edc-pouch/
├── images/            # Visuals of the pouch and contents
└── index.md           # Main documentation on the project
```

The `index.md` file details the rationale and supplies. Future updates include expanding the supply list and improving customization options.
